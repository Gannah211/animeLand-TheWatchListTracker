# ⬡ Anime Land

Anime Land is a simple and interactive **Anime Watch Tracker** web application that helps users organize and manage their anime journey in one place.

---

## ✨ Features

- 🔐 User Authentication (Sign Up / Sign In / Sign Out)
- 📺 Browse Anime List
- ➕ Add Anime to Watchlist
- 📊 Track Watch Status:
    - Plan to Watch
    - Currently Watching
    - Finished
    - Didn't Finish
- ⭐ Rate Anime (0 - 10)
- 🎬 Track Seasons (mark watched seasons)
- 💬 Add Comments
- ✏️ Update Comments
- 🗑️ Delete Comments
- 👤 User Profile Page

---

## 🧠 Concept

This project is designed to solve a common problem for anime fans:  
keeping track of what they’ve watched, what they’re currently watching, and what they plan to watch next — all in a clean and simple interface.

---
